# Vendored dataset

`values_tree.csv` and `values_frequencies.csv` are unmodified copies from
[Anthropic/values-in-the-wild](https://huggingface.co/datasets/Anthropic/values-in-the-wild),
revision `ab3f8507ad9513889edfbc5a9ec56b0072ae632c`.

The dataset is licensed under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/).
The hierarchy, cluster descriptions, value labels, and frequency measurements are attributed to
Anthropic's *Values in the Wild: Discovering and Analyzing Values in Real-World Language Model
Interactions* dataset and paper.
