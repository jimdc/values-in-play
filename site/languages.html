How Claude's values shift across languages

Anthropic measured how Claude's expressed values lean across four axes — Deference vs. Caution, Warmth vs. Rigor, Depth vs. Brevity, and Candor vs. Execution — in roughly 309,815 real conversations across the 20 most common languages on Claude.ai and 3 models. This page turns those profiles into something to play with: which languages cluster together, which axes divide them, and whether any of it rhymes with how people in different countries describe their own values.

Does it rhyme with human values data?

The Inglehart–Welzel cultural map plots countries by how their people answer the World Values Survey — traditional vs. secular-rational values, and survival vs. self-expression values. It measures humans, not Claude, and a language is not a country. Still: does the selected language's Claude profile land anywhere near its speakers' rough neighborhood on the human map?

Country positions were read by eye off a published chart image — there's no numeric table for this map, so treat two countries within about 0.2 of each other as roughly the same spot, not meaningfully different.

Why "anchor countries," and how were they picked?

A language isn't a country, so each language above is paired with one or a few "anchor countries" whose speakers stand in for it on this human-values map. Some languages get a single anchor (Hindi → India, Japanese → Japan); others get a small hand-picked set averaged together (English, Arabic, Spanish, Portuguese).

That averaging can blur real differences — the Spanish anchor, for instance, averages Spain with Mexico, Argentina, and Colombia, countries that sit in quite different neighborhoods of the human-values map. The country list behind each language's anchor is shown above whenever you select it, so the shortcut stays visible rather than hidden.

Read this comparison as a loose "does this rhyme?" prompt, not a claim that a language's speakers hold the values Claude expresses in that language, or that the two are measuring the same thing.

Where these numbers actually come from

Transcribed (the language, model, and axis-definition figures): Anthropic never published a data file for this study — the per-language and per-model numbers exist only as labels printed inside chart images in the article. Every figure here was copied by hand from those printed labels and cross-checked against the article's own callouts (like "Claude expresses the most warmth in Hindi and Arabic").

Estimated (the human-values map positions): no country-by-country coordinate table is published for the Inglehart–Welzel map either. Those positions were read by eye off a chart image at high resolution, so expect roughly ±0.1 to ±0.15 of error on each axis.

Treat every number on this page as a summary of a two-week 2026 sample and a single published chart image — not a definitive, durable measurement of a language, a model, or a country's values.

Value inference is subjective. This page is not a definitive assessment of Claude, language models, or any language's speakers.
