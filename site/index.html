A field guide to what Claude values

What is this?

Anthropic studied values expressed by Claude across roughly 308,000 real conversations, then organized 3,307 value labels into a four-level taxonomy. This field turns that research artifact into a place to wander.

Label size reflects how often a value appeared across conversations. Color marks one of five broad domains. Cluster context is generated from the dataset's hierarchy; the source does not provide definitions for individual leaf values.

Value inference is subjective, and the model-generated labels and descriptions may contain errors. This is not a definitive assessment of Claude or language models generally.
